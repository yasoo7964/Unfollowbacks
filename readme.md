Xで片思いさんをアンフォローする支援ツールです。
android用apkファイルです。

Xでたくさんの人をフォローしたいという場合、
Xには現在のフォロワー数✕１．ｎ倍までしかフォローもできないという制限があるので、
フォローバックして頂けてないかたは残念ながら古い順にアンフォローしていかなければなりません。

このアプリは
X公式アプリとの「フォロー中」画面と同じページを独自ブラウザで開き、
一番古い人まで検索し、
フォローバックしてくれているかたは赤表示且つアンフォローボタン無しで表示して視認性を上げる、
としただけのものです。
白く表示されたかたたちのみをアンフォローしていけば良いです。
